# ProjectWeatherCloth1 상세 매뉴얼

본 프로젝트는 고도화된 기상 분석과 개인화 엔진을 결합한 스마트 의류 추천 서비스입니다.

## 1. 주요 기능 가이드

### 1-1. 사용자 맞춤형 설정
- **민감도 조절**: 사용자의 '추위 민감도'와 '더위 민감도'를 1~5단계로 설정하여 추천의 정확도를 높일 수 있습니다.
- **지역 설정**: 기상청 API와 연동하여 해당 지역의 실시간 날씨 정보를 가져옵니다.

### 1-2. 지능형 추천 엔진 (Score-Engine)
- **CLO 지수 기반**: 의복의 보온 상태를 수치화한 CLO 개념을 도입했습니다.
- **복합 기상 분석**: 단순 온도뿐 아니라 습도, 풍속, 강수 확률을 연산에 포함합니다.
- **건강 데이터 반영**: 수면 시간, 최근 운동 여부 등에 따라 체온 보정값을 자동 계산합니다.

## 2. 전문가용 아키텍처 가이드

### Back-end (FastAPI)
- `backend/app.py`: 모든 요청의 입구이자 라우팅을 담당합니다.
- `backend/services/recommendation_service.py`: 본 프로젝트의 핵심 알고리즘이 위치합니다.

### Front-end (Streamlit)
- `frontend/app.py`: 백엔드 API와 통신하여 데이터를 시각화합니다.
- 대시보드 형태의 UI로 한눈에 정보를 파악할 수 있습니다.

## 3. 설치 및 문제 해결 (Troubleshooting)

- **백엔드 접속 오류**: `run.bat` 실행 시 백엔드 서버(8000포트)가 정상적으로 떴는지 확인하세요.
- **패키지 충돌**: 기존에 설치된 패키지와 충돌이 날 경우 `venv` 폴더를 삭제하고 `install.bat`을 재실행하세요.
